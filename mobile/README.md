# 📱 Mobile - Rotiserie & Pizza Moinești

## 🎯 Scop
Aplicații mobile native pentru platforma de comenzi online Rotiserie & Pizza Moinești.

## 📂 Structură

```
mobile/
├── README.md                 # Acest fișier
├── customer-ios/            # Aplicație clienți iOS
├── customer-android/        # Aplicație clienți Android
├── rider-ios/               # Aplicație rider iOS (viitor)
└── rider-android/           # Aplicație rider Android (viitor)
```

## 🚀 Tech Stack
- **React Native 0.73+**
- **Expo SDK 50+**
- **React Navigation**
- **Zustand** (state management)
- **React Query**

## 📋 Features Mobile-Specific
- Push notifications
- Background location tracking
- Camera pentru QR codes
- Offline mode pentru rideri
- Biometric auth

## ⏳ Status
**Proiect web-first** - Mobile va fi adaptat după finalizarea platformei web.
